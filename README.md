## Quick Start (Local)

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. That's it! No `npm install` required.

## Customization

- **Name/Bio**: Edit the text inside `index.html` (Hero and About sections).
- **Images**: 
    - Go to the `/assets/` folder (you need to create this if you cloned just the code).
    - Add your images: `avatar.png`, `project1.jpg`, etc.
    - Update the `<img src="...">` tags in `index.html` to point to your local files instead of the placeholder URLs.
- **Colors**: Open `styles.css` and edit the `:root` variables at the top.
